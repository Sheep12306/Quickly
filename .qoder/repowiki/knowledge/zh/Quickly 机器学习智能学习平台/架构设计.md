- **前端架构**：采用单页应用 (SPA) 模式，以 `src/App.tsx` 为状态管理中心，通过 `activeTab` 路由切换不同功能视图（学习、笔记、掌握度、路径、复习）。
- **后端集成**：使用 `server.ts` 作为 Express 中间件服务器，在开发模式下集成 Vite 中间件，生产模式下托管静态资源；同时提供 `/api/chat` 和 `/api/quiz` 接口，支持 Gemini API 调用或本地模拟器回退。
- **组件分层**：UI 拆分为布局组件（`Sidebar`, `RightPanel`）和功能模块组件（`ActiveQuiz`, `NotesChest`），通过 Props 与父组件状态交互。
- **样式方案**：使用 Tailwind CSS v4 进行原子化样式管理，配合 `motion/react` 实现交互动画。
